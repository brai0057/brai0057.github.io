HTML Tags I've Learned

<html>
The root element of an HTML page.
<head>
Contains meta-information about the document.
<body>
Defines the body of the document and contains its content.
<h1> to <h6>
Defines HTML headings, with <h1> being the largest and <h6> the smallest.
<p>
Defines a paragraph.
